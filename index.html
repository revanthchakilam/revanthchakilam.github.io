Ericsson Configuration Portfolio(ECP)

ECP is an intranet web application built on ASP as front end, JAVA as back end along with SYBASE database. We developed the online ecommerce portal which allows Ericsson users to select and configure various products, modelling them, add to cart and place an order. The entire portal works on integration of different systems. The products in our portal/database come from a SAP-CRM ONE system which are configured & order is placed in ECP and are sent to VERDI for tendering. There are various automatic jobs scheduled each day and each week. ECP portal is hosted on 6 (IBM WebSphere) WAS production servers using load balancing which optimizes any amount of load on to the system. The ASP code is deployed on the IIS servers. Continuous monitoring is scheduled 24/7 on all the production servers. Weekly meetings are scheduled with Ericsson TPM and BISO, Client requirements are discussed in it.
